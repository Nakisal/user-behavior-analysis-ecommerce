## 📊 项目目标
通过 1000 名用户的行为数据，识别高价值群体，提出精准运营建议。

## 🔍 分析框架
1. 用户画像：性别/年龄/地区与购买偏好的关系  
2. 行为洞察：活跃度、犹豫度、订阅影响  
3. 价值分群：RFM 模型识别四类用户  
4. 行动建议：个性化推荐与流失预警

## 📁 文件说明
- `原始数据.xlsx`：清洗后数据集  
- `活跃度动态分析.xlsx`：交互式分析  
- `核心结论.xlsx`：静态报告与可行性分析

## 💡 关键发现
- **39-45男性**男性购买力要强于女性，提供了约22%的总消费
- **订阅用户**订阅对于用户的购买行为的影响不大
- **不犹豫用户**（动态评定为不犹豫用户）总消费金额仅为其他用户的一半

## 🚀 行动建议
1.郊区对于图书类消费欲望较强，城市对于健康与美容消费能力较强，可以针对性投放广告
2.对于高活跃用户，购买欲望较强，对于其他活跃程度用户加强推荐算法，增强购买欲望
3.对于不同类别客户均呈现中低等活跃度，下一步改善措施应增强用户留存率，增加活跃度
4.在不同用户群中订阅比例无明显分化，进一步增强订阅转化率，目标为城市潜力用户
